## Why this matters

Real data is rarely flat. A user has an array of orders. Each order has an array of items. Each item has a price object with currency and amount. You need to reach into these nested structures, transform them, and build new ones.

## Arrays of objects

```javascript
const users = [
  { name: "Ada", age: 30 },
  { name: "Bob", age: 25 },
  { name: "Cal", age: 35 },
];

const names = users.map((u) => u.name);
console.log(names); // ["Ada", "Bob", "Cal"]

const older = users.filter((u) => u.age > 28);
console.log(older.length); // 2
```

This is the most common data shape in web development. API responses, database rows, and UI lists are all arrays of objects.

## Objects with arrays

```javascript
const course = {
  title: "Phase 1",
  modules: [
    { id: "1-1", title: "Variables" },
    { id: "1-2", title: "Control Flow" },
  ],
};

console.log(course.modules[0].title); // "Variables"
```

## Deep access with optional chaining

```javascript
const data = { user: { address: { city: "Austin" } } };
console.log(data.user.address.city); // "Austin"
console.log(data.user.phone?.number); // undefined — no crash
```

Without `?.`, accessing `data.user.phone.number` would throw because `phone` is `undefined`.

## Transforming nested data

```javascript
const orders = [
  { id: 1, items: [{ price: 10 }, { price: 20 }] },
  { id: 2, items: [{ price: 5 }] },
];

const totals = orders.map((order) => ({
  id: order.id,
  total: order.items.reduce((sum, item) => sum + item.price, 0),
}));

console.log(totals);
// [{ id: 1, total: 30 }, { id: 2, total: 5 }]
```

The pattern: `map` the outer array, `reduce` (or `map`/`filter`) the inner array, build a new shape.

## Flattening

```javascript
const nested = [[1, 2], [3, 4], [5]];
console.log(nested.flat()); // [1, 2, 3, 4, 5]

const tags = [{ tags: ["js", "ts"] }, { tags: ["rust"] }];
console.log(tags.flatMap((t) => t.tags)); // ["js", "ts", "rust"]
```

`flatMap` is `map` + `flat(1)` in one step.

## What you learned

- Real data is nested: arrays of objects, objects with arrays. Chain `map`, `filter`, `reduce` to transform it.
- `?.` safely navigates deep paths without crashing on missing properties.
- `flat()` and `flatMap()` collapse one level of nesting.
