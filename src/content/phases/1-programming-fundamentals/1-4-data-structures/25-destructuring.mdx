## Why this matters

Destructuring is syntax sugar for extracting values from arrays and objects. It replaces verbose `const name = user.name; const age = user.age;` with a single statement. Modern JavaScript uses it constantly.

## Object destructuring

```javascript
const user = { name: "Ada", age: 30, role: "engineer" };
const { name, age } = user;
console.log(name); // "Ada"
console.log(age); // 30
```

The variable names must match the property keys. You can rename with `:`:

```javascript
const { name: userName, role: userRole } = user;
console.log(userName); // "Ada"
```

## Default values

```javascript
const { name, country = "Unknown" } = { name: "Ada" };
console.log(country); // "Unknown"
```

The default applies when the property is `undefined`.

## Array destructuring

```javascript
const [first, second, third] = [10, 20, 30];
console.log(first); // 10
console.log(third); // 30
```

Skip elements with empty slots:

```javascript
const [, , last] = [1, 2, 3];
console.log(last); // 3
```

## Rest in destructuring

```javascript
const { name, ...rest } = { name: "Ada", age: 30, role: "engineer" };
console.log(name); // "Ada"
console.log(rest); // { age: 30, role: "engineer" }

const [head, ...tail] = [1, 2, 3, 4];
console.log(head); // 1
console.log(tail); // [2, 3, 4]
```

## In function parameters

```javascript
function greet({ name, greeting = "Hello" }) {
  return `${greeting}, ${name}!`;
}

console.log(greet({ name: "Dustin" })); // "Hello, Dustin!"
```

This is the standard pattern for options objects.

## What you learned

- **Object destructuring** extracts named properties: `const { a, b } = obj`.
- **Array destructuring** extracts by position: `const [x, y] = arr`.
- Use `...rest` to collect leftover properties or elements.
