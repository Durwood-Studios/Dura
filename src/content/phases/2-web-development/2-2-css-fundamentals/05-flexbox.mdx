## Why this matters

Flexbox is the CSS layout model you'll use most often. It handles one-dimensional layouts — rows or columns — with built-in alignment, distribution, and wrapping. Before flexbox, centering a div vertically required hacks. Now it takes three lines.

## Flex containers

Adding `display: flex` to an element turns it into a **flex container**. Its direct children become **flex items**.

```css
.container {
  display: flex;
}
```

By default, flex items line up in a row, left to right, and shrink to fit their content. Every flex container has a **main axis** (the direction items flow) and a **cross axis** (perpendicular).

### Container properties

```css
.container {
  display: flex;

  /* Direction of the main axis */
  flex-direction: row; /* Default: left to right */
  flex-direction: column; /* Top to bottom */

  /* How items are distributed along the main axis */
  justify-content: flex-start; /* Pack items to the start (default) */
  justify-content: center; /* Center items */
  justify-content: space-between; /* First item at start, last at end, equal gaps */
  justify-content: space-around; /* Equal space around each item */
  justify-content: space-evenly; /* Equal space between and around items */

  /* How items align on the cross axis */
  align-items: stretch; /* Default: items stretch to fill container height */
  align-items: center; /* Center items vertically (in a row layout) */
  align-items: flex-start; /* Align to top */
  align-items: flex-end; /* Align to bottom */
  align-items: baseline; /* Align by text baseline */

  /* Space between items */
  gap: 16px; /* Equal gap in both directions */
  gap: 16px 24px; /* Row gap, then column gap */

  /* Whether items wrap to new lines */
  flex-wrap: nowrap; /* Default: all items on one line */
  flex-wrap: wrap; /* Items wrap to new lines when they don't fit */
}
```

The most common pattern — centering something horizontally and vertically:

```css
.centered {
  display: flex;
  justify-content: center;
  align-items: center;
  min-height: 100vh;
}
```

### Building a navigation bar

```css
.navbar {
  display: flex;
  justify-content: space-between;
  align-items: center;
  padding: 0 24px;
  height: 64px;
}
```

```html
<nav class="navbar">
  <div class="logo">DURA</div>
  <ul class="nav-links">
    <li><a href="/learn">Learn</a></li>
    <li><a href="/dictionary">Dictionary</a></li>
  </ul>
</nav>
```

The logo pushes left, the links push right, and `align-items: center` vertically centers both.

## Flex items

Individual flex items have their own properties that control how they grow, shrink, and size themselves.

```css
.item {
  /* How much this item grows relative to siblings */
  flex-grow: 0; /* Default: don't grow */
  flex-grow: 1; /* Take up available space equally */

  /* How much this item shrinks when space is tight */
  flex-shrink: 1; /* Default: shrink equally */
  flex-shrink: 0; /* Don't shrink — maintain size */

  /* The initial size before growing/shrinking */
  flex-basis: auto; /* Default: use content size or width */
  flex-basis: 200px; /* Start at 200px */

  /* Shorthand: grow shrink basis */
  flex: 1; /* Same as: flex: 1 1 0% — grow equally */
  flex: 0 0 300px; /* Don't grow, don't shrink, stay at 300px */

  /* Override cross-axis alignment for this item only */
  align-self: center;

  /* Reorder visually (not in the DOM) */
  order: -1; /* Move to the front */
}
```

### Building a card layout

```css
.card-grid {
  display: flex;
  flex-wrap: wrap;
  gap: 24px;
}

.card {
  flex: 1 1 300px; /* Grow and shrink, starting at 300px */
  padding: 24px;
  background: rgba(255, 255, 255, 0.04);
  border: 1px solid rgba(255, 255, 255, 0.08);
  border-radius: 12px;
}
```

With `flex: 1 1 300px`, each card starts at 300px wide. If there's extra space, they grow equally. If the container is too narrow, they wrap to a new row. This creates a responsive layout without media queries.

### The flex shorthand

The `flex` shorthand is the preferred way to set item sizing. Common values:

```css
.fill {
  flex: 1;
} /* Grow to fill, shrink equally, basis 0% */
.fixed {
  flex: 0 0 250px;
} /* Never grow or shrink, always 250px */
.fluid {
  flex: 1 1 auto;
} /* Grow and shrink from content size */
```

## Try it

<SandboxExercise language="javascript">
{`// Simulate how flex-grow distributes remaining space.
// Given a container width and an array of items
// (each with flexBasis and flexGrow), calculate each item's final width.

function flexLayout(containerWidth, items) {
const totalBasis = items.reduce((sum, item) => sum + item.flexBasis, 0);
const remainingSpace = containerWidth - totalBasis;
const totalGrow = items.reduce((sum, item) => sum + item.flexGrow, 0);

return items.map(item => {
if (totalGrow === 0) return item.flexBasis;
const growShare = (item.flexGrow / totalGrow) \* remainingSpace;
return item.flexBasis + growShare;
});
}

// Container is 600px with 3 items
const result = flexLayout(600, [
{ flexBasis: 100, flexGrow: 1 },
{ flexBasis: 100, flexGrow: 2 },
{ flexBasis: 100, flexGrow: 1 },
]);

// Remaining space: 600 - 300 = 300. Grow ratio 1:2:1
console.log(result[0] === 175); // 100 + 75
console.log(result[1] === 250); // 100 + 150
console.log(result[2] === 175); // 100 + 75
console.log(result[0] + result[1] + result[2] === 600);`}

</SandboxExercise>

## What you learned

- `display: flex` creates a one-dimensional layout where `justify-content` distributes items along the main axis and `align-items` aligns them on the cross axis
- Flex item properties (`flex-grow`, `flex-shrink`, `flex-basis`) control how each item claims available space, with the `flex` shorthand as the preferred syntax
- Combining `flex-wrap: wrap` with a `flex-basis` value creates responsive layouts that reflow without media queries
