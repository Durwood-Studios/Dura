## Why this matters

AI APIs charge per token. A sloppy prompt can cost 10x more than a tight one. Understanding token economics is the difference between an AI feature that's profitable and one that bankrupts your budget.

## What a token costs

Pricing is per million tokens, split between input and output:

```text
Claude Sonnet 4:  $3 / 1M input,  $15 / 1M output
Claude Opus 4:    $15 / 1M input, $75 / 1M output
GPT-4o:           $2.50 / 1M input, $10 / 1M output
GPT-4o-mini:      $0.15 / 1M input, $0.60 / 1M output
```

Output tokens cost 3-5x more than input tokens because input is processed in parallel (one forward pass) while output is generated sequentially (one pass per token).

## Estimating costs before you run

```javascript
function estimateCost(inputText, expectedOutputTokens, model) {
  const pricing = {
    "claude-sonnet": { input: 3, output: 15 },
    "gpt-4o": { input: 2.5, output: 10 },
    "gpt-4o-mini": { input: 0.15, output: 0.6 },
  };

  const rates = pricing[model];
  const inputTokens = Math.ceil(inputText.length / 4);
  const inputCost = (inputTokens / 1_000_000) * rates.input;
  const outputCost = (expectedOutputTokens / 1_000_000) * rates.output;

  return {
    inputTokens,
    outputTokens: expectedOutputTokens,
    totalCost: inputCost + outputCost,
  };
}
```

## Cost optimization strategies

**1. Choose the right model.** GPT-4o-mini at $0.15/M input is 20x cheaper than Claude Sonnet. If your task is simple classification, use the smallest model that works.

**2. Shorter prompts.** Every token in your system prompt is charged on every request. A 500-token system prompt across 100K daily requests costs real money.

**3. Cache repeated context.** Anthropic's prompt caching reduces repeated input token costs by 90%. If 80% of your prompt is the same across requests, cache it.

**4. Set max_tokens wisely.** Don't set max_tokens to 4096 when you expect 100 tokens. Lower limits mean the model stops sooner.

**5. Model routing.** Use a cheap model for easy tasks and a powerful model for hard ones. A classifier routes each request.

## Context window math

```text
Context window = input tokens + output tokens

If your context window is 200K tokens and your input is 180K,
you have at most 20K tokens for the response.
```

Exceeding the window causes an error. Always leave headroom for the response.

## What you learned

- AI APIs charge per token. Output costs 3-5x more than input due to sequential generation.
- The biggest cost lever is model selection. Use the cheapest model that works for the task.
- Estimate costs before running. Cache repeated context. Set max_tokens conservatively.
