## Why this matters

Every photo on your phone, every word in this lesson, every dollar in your bank account is, at the bottom, a long string of <VocabTooltip slug="bit">bits</VocabTooltip>. If you understand binary, you understand the floor that everything else stands on.

You will not need to do binary math in your day job. You will need to understand it the day a bug stops making sense, and the answer turns out to be that the computer was counting in twos when you assumed it was counting in tens.

## A switch and a number

Imagine a single light switch. It is either off or on. That is one bit of information.

We write "off" as `0` and "on" as `1`. That is the entire alphabet a computer speaks.

```javascript
// In JavaScript, you can ask for any number's binary form.
console.log((5).toString(2)); // "101"
console.log((10).toString(2)); // "1010"
```

> A **bit** is a single 0 or 1. Eight bits make a **byte**, which can represent 256 different values. That is enough to cover every letter on a keyboard, with room to spare.

## Counting in twos

In the number system you grew up with — base 10 — each column is worth ten times the one to its right: ones, tens, hundreds, thousands.

Binary works the same way, but each column is worth two times the one to its right: ones, twos, fours, eights, sixteens.

So `1010` in binary is:

```text
1 × 8 = 8
0 × 4 = 0
1 × 2 = 2
0 × 1 = 0
        ─
        10
```

That is the number ten, written using only switches.

## From numbers to letters

If a byte can hold 256 values, you can map each value to a letter. That is exactly what <VocabTooltip slug="algorithm">ASCII</VocabTooltip> does. The capital letter `A` is the number 65. Lowercase `a` is 97. The digit `0` (as a character) is the number 48.

```javascript
// Three different ways to look at the same letter.
console.log("A".charCodeAt(0)); // 65
console.log((65).toString(2)); // "1000001"
console.log(String.fromCharCode(65)); // "A"
```

Pictures, sound, and video all work the same way. They are just numbers, stored as bits, decoded by software that knows how to interpret them.

## What you learned

- Computers store everything as **bits** — switches that are either 0 or 1.
- Eight bits make a **byte**, which can represent 256 distinct values.
- ASCII is one mapping from those values to letters; pictures and sound use other mappings, but the underlying bits are the same.
