## Step 1: Install a Code Editor

The first tool every developer needs is a **code editor**. We recommend Visual Studio Code (VS Code) — it's free, fast, and has a massive ecosystem of extensions.

**What to do:**

1. Go to [code.visualstudio.com](https://code.visualstudio.com)
2. Download the installer for your operating system
3. Run the installer with default settings
4. Launch VS Code

## Step 2: Set Up Your Terminal

Your terminal is where you'll run commands, install packages, and interact with git.

**macOS/Linux:** You already have Terminal or you can install [iTerm2](https://iterm2.com).

**Windows:** Install [Windows Terminal](https://aka.ms/terminal) from the Microsoft Store, then enable WSL2:

```bash
wsl --install
```

After restarting, you'll have a full Linux environment inside Windows.

## Step 3: Install Node.js

Node.js lets you run JavaScript outside the browser. We recommend using **nvm** (Node Version Manager) so you can switch versions easily.

**Install nvm:**

```bash
# macOS/Linux
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.7/install.sh | bash

# Then install Node
nvm install --lts
nvm use --lts
```

**Verify installation:**

```bash
node --version
npm --version
```

You should see version numbers (e.g., `v20.11.0` and `10.2.0`).

## Step 4: Install Git

Git is the version control system used by virtually every software project.

```bash
# macOS (comes pre-installed, or via Homebrew)
brew install git

# Ubuntu/Debian
sudo apt install git

# Windows (WSL)
sudo apt install git
```

**Configure your identity:**

```bash
git config --global user.name "Your Name"
git config --global user.email "your.email@example.com"
```

<Checkpoint
  id="git-configured"
  label="Git Configured"
  status="active"

  answer="git config user.name"

  Make sure git is installed and your identity is set.
</Checkpoint>

## Step 5: Install Essential VS Code Extensions

Open VS Code and install these extensions (Ctrl+Shift+X to open the extensions panel):

| Extension      | Why                                              |
| -------------- | ------------------------------------------------ |
| **ESLint**     | Catches JavaScript/TypeScript errors as you type |
| **Prettier**   | Auto-formats your code on save                   |
| **GitLens**    | See who changed each line and when               |
| **Error Lens** | Shows errors inline, right next to the problem   |

**Enable format-on-save:**

1. Open Settings (Ctrl+,)
2. Search for "format on save"
3. Check the box

<Checkpoint id="extensions-installed" label="Extensions Installed" status="active">
  Install at least ESLint and Prettier, then enable format-on-save.
</Checkpoint>

## Step 6: Create Your First Project

Let's verify everything works by creating a small project:

```bash
mkdir my-first-project
cd my-first-project
npm init -y
```

Open it in VS Code:

```bash
code .
```

Create a file called `index.js`:

```javascript
const greeting = "Hello, developer!";
console.log(greeting);
console.log(`Node.js version: ${process.version}`);
```

Run it:

```bash
node index.js
```

## Step 7: Set Up a GitHub Account

GitHub is where you'll store your code, collaborate with others, and build your portfolio.

1. Go to [github.com](https://github.com) and create an account
2. Set up SSH keys for passwordless authentication:

```bash
ssh-keygen -t ed25519 -C "your.email@example.com"
cat ~/.ssh/id_ed25519.pub
```

3. Copy the output and add it to GitHub: Settings → SSH and GPG keys → New SSH key

**Test the connection:**

```bash
ssh -T git@github.com
```

You should see: "Hi username! You've successfully authenticated."
